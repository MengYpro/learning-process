1. Arduino安装
        安装在~/Development/Arduino
        项目和其他内容保存在~/Development/Arduino/Arduino-Process
        （以后有可能碰上一起软件或者功能会硬在～里找Arduino保存点，需要注意）

2. 给arduino加别名
	因为官网下载下来的是AppImage,加别名比较方便
	vim ~/.bashrc
	添加 aliad arduino="~/Development/Arduino/blablabla.AppImage"

3. htop查看电脑进程
	sudo apt install htop
	终端直接输入htop打开
	输入htop -u username过滤掉系统进程

4. build-essential
	提供GCC/G++用于C和C++的编译工具，Make，Debian（UBUNTU的基础）

5. python3-dev,python3-pip,python3-venv
	python的头文件，软件包；包管理；python虚拟环境

6. curl用于跟网络交互，从服务区获取或上传数据,wget类似，只是更专注于下载

7. neofetch用于图形化显示电脑系统，软件硬件信息
	安装后直接在终端输入neofetch即可

8. sudo apt install cmake clang clang-tools
	用于C++和CMake跨平台的构建系统生成器

9. 压缩软件
	tar和gzip系统自带
	zip和unzip用于和win交互

10. ubuntu-restricted-extras
	安装一些多媒体功能以及有专利限制的内容，需同意协议

11. sudo apt install gnome-tweaks   GNOME调整工具

12. sudo apt install fonts-noto-cjk fonts-noto-color-emoji中英文字体

13. Visual Studio Code
	终端：code

14. sudo apt install libreoffice办公软件

15. sudo apt install steam

	Proton/Wine: sudo apt install wine winetricks
